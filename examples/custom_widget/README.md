# Custom Widget Extension Example

This example demonstrates a minimal extension package shape:

- custom widget: `UnifiedUiObservability.Extension.Widgets.MetricBadge`
- `UnifiedIUR.Element` implementation for the custom widget
- custom renderer: `UnifiedUiObservability.Extension.Renderers.Terminal`

Source file:

- `examples/custom_widget/lib/unified_ui_observability/extension.ex`
