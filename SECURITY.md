# Security Policy

## Supported Versions

The latest release on `main` is considered supported for security updates.

## Reporting a Vulnerability

Please do not open public issues for security vulnerabilities.

Report vulnerabilities by email to: **pcharbon70@gmail.com**

Include:

- A description of the vulnerability
- Steps to reproduce
- Potential impact
- Any suggested remediation

We will acknowledge receipt within 5 business days and provide status updates as triage progresses.
